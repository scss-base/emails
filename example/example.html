SCSS Base Emails

Use SCSS Base Emails to design responsive HTML emails that work in any email client.

Examples

Button

Buttons in HTML email are usually simple table structures, with an anchor inside.
Button

Sizing

Buttons can be made larger or smaller by adding the class .tiny, .small, or .large to a button's outer <table>.

Expanded

To create an expanded button (full width of it's container), add the class .expanded to the outer <table> of the button, and wrap a <center> tag around the <a>.

Add the .expanded class to the <table class="button"> tag. If you want the button to be expanded on small only, add the .small-expanded.
Expanded Button

Coloring

Change the background color of a button by adding the class .secondary, .success, .warning, or .danger to the outer <table> (or the <button> tag in Inky HTML).
Secondary Button
Success Button
Warning Button
Danger Button

Radius and Round

Creating buttons with a radius or rounded edges (like a pill) can be achieved by adding the .radius or .rounded class to a button.

Note - border-radius is not supported on Outlook 2000-2013, Yahoo! Mail, and Android 4+ (Gmail)

Note - In order to create .radius and .rounded buttons, the border needs to be removed.
